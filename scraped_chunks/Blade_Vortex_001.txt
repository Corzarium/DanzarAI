Maximum blades: Up to 10 blade stacks can be active at a time. Each stack has its own duration. Hit interval: When the blade is first spawned, the skill will deal a hit after the hit interval duration (0.6 seconds with one blade). Multiple copies: You can only have one Blade Vortex effect at any one time. Casting Blade Vortex with another copy of the gem will remove all active stacks and start a new Blade Vortex effect. Multiple casts: Using Blade Vortex while you have no Blade Vortex effect will create a new Blade Vortex effect with all the "spell cast" modifiers baked into that Blade Vortex effect. Any subsequent uses of Blade Vortex will only add a stack to the currently active effect.
